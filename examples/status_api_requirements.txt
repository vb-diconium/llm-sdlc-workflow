Build a minimal single-endpoint status API.

Requirements:
1. GET /status → returns JSON: { "status": "ok", "service": "status-api", "timestamp": "<ISO 8601 datetime>" }
2. GET /health → returns JSON: { "status": "ok" }

Technology:
- Python with FastAPI, Uvicorn
- No database, no authentication, no BFF, no frontend
- Stateless service

Non-functional:
- Backend only, single service on port 8080
- Multi-stage Dockerfile (builder + runtime image)
- docker compose up starts it with one command
